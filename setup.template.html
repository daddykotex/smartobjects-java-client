Get the client library

The client library is available on maven.

You can get started quickly by adding the following to your pom file:

    
<dependency>
    <groupId>com.mnubo</groupId>
    <artifactId>java-sdk-client</artifactId>
    <version>1.12.0</version>
</dependency>

Learn more on: GitHub.

Create a client instance

To create an instance, you need to import the following classes:

    
import com.mnubo.java.sdk.client.services.MnuboSDKFactory;
import com.mnubo.java.sdk.client.spi.MnuboSDKClient;

After that, you can create the client:

    
final String host = "<%= hostname %>;
final String clientKey = "<%= clientKey %>";
final String clientSecret = "<%= clientSecret %>";

final MnuboSDKClient mnuboClient = MnuboSDKFactory.getClient(host , clientKey, clientSecret);
